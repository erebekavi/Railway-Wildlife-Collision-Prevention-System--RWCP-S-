## 1. Front-end

- **React.js**
    - Real-time dashboard for monitoring
    - Displays camera feeds and alerts
    - User interface for control center
- HTML5, CSS3, JavaScript

## 2. Back-end

- **Python**
- **Django**
    - REST API development
    - System integration (AI + IoT + front-end)
    - Alert management
    - Authentication & security
    - Admin panel for monitoring

## 3. Database

- **SQLite** (for prototype/demo)
- (Optional for scaling: PostgreSQL / MongoDB)

## 4. Computer Vision

- **OpenCV**
    - Video stream processing
    - Image enhancement and preprocessing
    - Frame extraction for AI models

## 5. AI / Machine Learning

- **TensorFlow**
    - Training deep learning models
- **Scikit-learn**
    - Data preprocessing
    - Feature engineering

## 6. AI Models & Architectures

- CNN (Convolutional Neural Network)
    - Object detection (humans, animals, obstacles)
- LSTM (Long Short-Term Memory)
    - Time prediction (movement, behavior)
- Hybrid CNN-LSTM
    - Combined spatial and temporal analysis

## 7. Edge Computing & Deployment

- **TensorFlow Lite (TFLite)**
    - Edge deployment of trained models
    - Real-time inference on device
- Quantization (INT8)
    - Reduces model size and improves speed
- XNNPACK Delegate
    - Optimized CPU inference
- Edge Inference
    - Low latency processing without cloud dependency

## 8. Communication & Integration

- 5G Connectivity
    - Real-time data transmission
- REST APIs / WebSockets
    - Communication between frontend, backend, and IoT devices

## 9. Data Processing Tools

- Pandas – Data manipulation
- NumPy – Numerical computation
- Scikit-learn – Preprocessing
- OpenCV – Image preprocessing

## Final Summary

- React.js → Frontend dashboard
- Django (Python) → Backend system
- SQLite → Database (prototype)
- OpenCV → Computer vision
- TensorFlow → Model training
- TensorFlow Lite → Edge deployment
- CNN, LSTM, CNN-LSTM → AI models
